# Weights & data assets

Large binary assets are **not** stored in git. Fetch them here before running the
engine at full strength. The engine finds these automatically (relative to the
executable), or point at them with the `EvalFile` / `SyzygyPath` UCI options.

## Stockfish HalfKP NNUE network (required for full strength)

```bash
curl -L -o nn-62ef826d1a6d.nnue \
  https://github.com/official-stockfish/networks/raw/master/nn-62ef826d1a6d.nnue
```

Without it the engine falls back to a (much weaker) handcrafted evaluation.

## Syzygy endgame tablebases (optional — perfect endgame play)

Download the 3-4-5-man tables (~939 MB) into `weights/syzygy/`:

```bash
mkdir -p syzygy && cd syzygy
base="http://tablebase.sesse.net/syzygy/3-4-5/"
curl -sL "$base" | grep -oE '[A-Za-z]+v[A-Za-z]+\.rtb[wz]' | sort -u | \
  while read -r f; do [ -s "$f" ] || curl -sL "$base$f" -o "$f"; done
```

With these present the engine plays ≤5-man endgames perfectly (DTZ-optimal).
